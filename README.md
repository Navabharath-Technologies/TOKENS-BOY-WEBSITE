# Token Boy Website

**Easy Tokens for Indian Clinics by Navabharath Technologies**

## Overview

Token Boy is a streamlined, digitized queue and scheduling system designed for modern healthcare. This website showcases the solution from the patient/user's perspective, highlighting the benefits of the system for clinic visitors.

## Features

- **Modern, Responsive Design**: Beautiful orange color scheme with smooth animations
- **Complete Information Architecture**:
  - Hero Section with company branding
  - Main Pain Points section highlighting problems with traditional systems
  - Our Solution section showcasing Token Boy features
  - How It Works section explaining the workflow
  - Benefits section demonstrating time management improvements
  - Demo Request Form for interested clinics
  - Contact Information
  
- **Interactive Elements**:
  - Smooth scroll navigation
  - Animated scroll reveals
  - Hover effects on cards
  - Form validation
  - Mobile-responsive hamburger menu

## Technology Stack

- **HTML5**: Semantic markup
- **CSS3**: Modern styling with gradients, animations, and flexbox/grid layouts
- **JavaScript**: Vanilla JS for interactivity
- **Font Awesome**: Icons
- **Google Fonts**: Poppins and Playfair Display fonts
